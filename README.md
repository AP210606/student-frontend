# 🛒 E-Commerce Web Application

🔗 **Live Demo:** https://candid-druid-cee9e2.netlify.app/

---

## 📌 Project Overview
This is a Full Stack E-Commerce Web Application built using modern web technologies. It includes user authentication, product listing, cart functionality, and seller dashboard features.

---

## 🚀 Features
- User Registration & Login
- Protected Routes (Authentication-based access)
- Product Listing Page
- Add to Cart functionality
- Seller Dashboard
- Responsive UI

---

## 🛠️ Tech Stack
- **Frontend:** React.js, Tailwind CSS  
- **State Management:** Redux  
- **Routing:** React Router  
- **Backend:** Node.js, Express.js *(if applicable)*  
- **Database:** MongoDB *(if applicable)*  

---

## 📂 Project Structure
- `components/` → Reusable UI components  
- `pages/` → Application pages (Home, Login, Products, etc.)  
- `redux/` → State management  
- `services/` → API handling
